# 📰 News Application (Python Project)

A simple command-line based News Application built using Python.  
This project reads news data from a text file and displays:

- 🌍 International News
- 🇮🇳 Indian News
- 📄 News Descriptions

The application uses basic Python concepts such as:
- Functions
- Lists
- File Handling using `os`
- Conditional Statements
- Loops
- User Input Handling

---

# 📌 Features

✅ Display International News Headlines  
✅ Display Indian News Headlines  
✅ Read News Description  
✅ Menu Driven Program  
✅ Simple and Beginner Friendly  
✅ Uses Text File Instead of API/JSON Library  

---

# 🛠️ Technologies Used

- Python 3
- OS Module
